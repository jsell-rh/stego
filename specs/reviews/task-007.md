# Review: Task 007 — Slot/Fill Proto Contract & Interface Generation

## Findings

- [x] [process-revision-complete] **`GenerateInterface` omits the mandatory `// Code generated by stego. DO NOT EDIT.` header.** The spec ("Generated Code Structure", rule 1) requires all generated files carry this header. The `gen` package already defines `gen.Header` and `gen.File.Bytes()` prepends it (established in task-006), but `GenerateInterface` returns raw Go source without it. The generated output must include the header so downstream consumers (drift detection, `stego drift`) can identify generated files.
